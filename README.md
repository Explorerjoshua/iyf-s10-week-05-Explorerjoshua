# IYF Season 10 - Week 05 Assignment

## 📌 Description

This project focuses on building interactive web pages using JavaScript by working with the DOM (Document Object Model) and handling user events. It demonstrates how to select, traverse, and manipulate elements dynamically, as well as respond to user interactions such as clicks, keyboard input, and form submissions.

---

## 🚀 Deliverables

### ✅ Interactive To-Do List Application

A fully functional to-do list app with:

* Add, delete, and complete tasks
* Filter tasks (All, Active, Completed)
* Task counter
* Clear completed tasks
* ✨ Edit tasks (double-click)
* ✨ Data persistence using localStorage
* ✨ Dark mode toggle (saved preference)

---

### ✅ DOM Manipulation (Lesson 9)

* Element selection methods
* DOM traversal (parent, children, siblings)
* Content updates (`textContent`, `innerHTML`)
* Attribute handling
* Style and class manipulation
* Dynamic element creation/removal

---

### ✅ Event Handling (Lesson 10)

* Click, input, submit, and keyboard events
* Event object usage (`target`, `preventDefault`)
* Event bubbling and delegation
* Form handling and validation

---
